# ⏳ Time-Coin

**O "Tinder" das Micro-Habilidades Locais.**

## 🚀 Sobre o Projeto
O Time-Coin é uma aplicação desenhada para combater o alto custo de vida e reconstruir o sentido de comunidade. A plataforma permite a troca de tempo e pequenos serviços locais (ex: montar um móvel, dar uma aula de inglês, arranjar um PC) utilizando um sistema de créditos virtuais em vez de dinheiro real.

## ✨ Funcionalidades Principais (MVP)
* **Geolocalização:** Encontra pessoas na tua zona dispostas a ajudar ou a precisar de ajuda.
* **Sistema de Créditos (Time-Coins):** 1 hora do teu tempo = X moedas. Uma economia justa baseada em esforço e não em dinheiro.
* **Matchmaking Inteligente:** Sugestões baseadas nas habilidades que ofereces e nas que procuras.
* **Assistente IA:** Integração com IA para ajudar a descrever as tarefas de forma rápida e clara.

## 🛠️ Tecnologias Utilizadas
* **Frontend:** React, Vite, Tailwind CSS, Motion
* **Backend/DB:** Express / Firebase (Autenticação e Firestore)
* **IA:** API Gemini (@google/genai)

## 🚀 Como testar localmente
1. Clona este repositório.
2. Corre `npm install` para instalar as dependências.
3. Corre `npm run dev` para iniciar o servidor local.
